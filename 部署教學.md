# 📱 讓手機能玩:免費部署到 Render 教學

整個過程不用碰命令列,大概 10 分鐘。部署完會拿到一個網址(像 `https://draw-guess-xxxx.onrender.com`),手機開這個網址就能多人玩。

---

## 你需要先有的東西
1. 一個 **GitHub 帳號**(免費,github.com 註冊)
2. 一個 **Render 帳號**(免費,render.com 用 GitHub 登入,不用刷卡)

---

## 步驟一:把程式放上 GitHub

最簡單的方法(網頁操作,免安裝 git):

1. 到 github.com,右上角 **+** → **New repository**
2. 取名 `draw-guess`,選 **Public**(或 Private 都行),按 **Create repository**
3. 進到剛建好的空 repo,點 **uploading an existing file**
4. 把這個資料夾裡的檔案**全部拖進去**:
   - `server.js`
   - `package.json`
   - `package-lock.json`
   - `render.yaml`
   - `.gitignore`
   - `public/` 整個資料夾(裡面的 index.html)
   - README.md(可選)
   ⚠️ **不要上傳 `node_modules`**(很大又用不到,Render 會自己裝)
5. 按 **Commit changes**

> 如果你會用 git,直接 `git init && git add . && git commit -m "init" && git push` 更快。

---

## 步驟二:在 Render 部署

1. 到 render.com,用 GitHub 登入
2. 右上 **New +** → **Web Service**
3. 選 **Build and deploy from a Git repository** → **Next**
4. 找到你的 `draw-guess` repo，按 **Connect**
5. 設定欄位(大部分會自動帶,確認一下就好):
   - **Name**: `draw-guess`(會變成網址的一部分)
   - **Region**: 選 **Singapore**(離台灣最近,最順)
   - **Runtime**: Node
   - **Build Command**: `npm install`
   - **Start Command**: `node server.js`
   - **Instance Type**: 選 **Free**
6. 按 **Create Web Service**
7. 等 2~3 分鐘,看到 log 出現 `🎨 你畫我猜跑起來了` 就成功了
8. 上方那個 `https://draw-guess-xxxx.onrender.com` 就是你的遊戲網址

---

## 步驟三:手機開來玩

1. 手機瀏覽器開那個網址
2. 輸入暱稱 + 隨便一組房號(例如 `abc`),進房
3. 點右上 **邀請 🔗**,直接分享到 LINE 給朋友
4. 朋友開連結、輸入**同一個房號**就進同一房
5. 房主按 **開始遊戲**,就能輪流畫、搶答了!

---

## 要注意的兩件事

- **第一次開會等約 1 分鐘**:免費方案閒置 15 分鐘會休眠,有人連進來才喚醒。喚醒後就順了。想完全不休眠要升級付費($7/月起),測試不用。
- **改了程式想更新**:重新上傳檔案到 GitHub(或 git push),Render 會自動重新部署。

---

## 想自己電腦先試跑?

裝好 Node 18+ 後,在資料夾裡:
```bash
npm install
node server.js
```
然後同一個 wifi 下,手機開 `http://你電腦的內網IP:3000`(例如 `http://192.168.1.5:3000`)也能連。查 IP:Mac 在「系統設定→網路」,Windows 用 `ipconfig`。
