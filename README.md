# 你畫我猜 Draw & Guess(即時多人 / 簡繁皆可作答)

## 怎麼跑

需要 Node 18+。在你的伺服器上:

```bash
npm install
node server.js
```

預設開在 `http://localhost:3000`。要改 port:`PORT=8080 node server.js`。

部署到正式機(讓朋友連進來):
- 確保防火牆/安全群組開放你設定的 port
- 建議用 `pm2 start server.js` 或 systemd 常駐
- 想用網域 + HTTPS,前面掛 Nginx 反向代理到 3000 即可(WebSocket 記得開 `proxy_set_header Upgrade`)

## 怎麼玩

1. 每個人開網頁,輸入暱稱 + **同一組房間代碼**就會進同一房
2. 房主按「開始遊戲」,系統隨機輪流指定畫家
3. 畫家從 3 個詞裡選一個來畫,其他人在聊天框打字搶答
4. **繁體、簡體都能答**(系統自動正規化比對,忽略標點空白大小寫)
5. 越早答對分數越高;畫家每被人猜中也加分

## 功能

- 隨機出題、畫家選詞(15 秒沒選自動選第一個)
- 即時同步畫布(中途加入會自動補上目前畫面)
- 畫筆 6 色、3 種粗細、填色(油漆桶)、復原、清空
- 難度切換(簡單/普通/困難)+ 自訂詞庫
- 計時、計分、排行,畫家落跑自動跳題

## 自訂

- 改詞庫:`server.js` 最上面的 `DEFAULT_WORDS`
- 改回合秒數:`makeRoom()` 裡的 `roundLength`(預設 80 秒)
